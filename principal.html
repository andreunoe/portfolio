DISEÑO GRÁFICO:
Como diseñador gráfico, creo identidades visuales, logotipos, elementos gráficos, banners y más, utilizando herramientas como Adobe Illustrator, InDesign o Photoshop.
Por ejemplo: Un cartel promocional para un evento cultural, una infografía sobre el cambio climático, el logotipo de una marca...
DISEÑO 3D:
En diseño 3D, desarrollo modelos y animaciones en Maya o Fusion, entre otros, aplicando técnicas de modelado y renderizado.
Por ejemplo: Modelado de un personaje 3D para un cortometraje o modelado de un determinado instrumento antes de su fabricación.
DESARROLLO WEB Y APP:
Desarrollo webs y aplicaciones funcionales con HTML, CSS y JavaScript en la parte frontend y PHP y MySQL en la parte backend. Además, mi trabajo consiste en realizar igualmente todo tipo de wireframes antes del desarrollo de las plataformas y de adaptar estas a un diseño responsive una vez iniciado el proceso.
Por ejemplo: Un sitio web interactivo para un comercio o una plataforma para una aplicación.
DISEÑO ESPECIAL:
Con diseño especial me refiero a un tipo de diseño orientado hacia determinadas técnicas de fabricación como el vinilado o el corte láser, además de los diferentes materiales utilizados para ello como la madera o el metal.
Por ejemplo: Diferentes juegos de mesa realizados mediante corte láser en madera.
DISEÑO ORIENTADO:
El diseño orientado busca soluciones específicas para cada proyecto, adaptando interfaces y experiencias.
Por ejemplo: Prototipos UX/UI para una app en Figma.
VÍDEOS:
Produzco y edito vídeos creativos con Premiere y After Effects, incluyendo cortos, videoclips, anuncios y más.
Por ejemplo: Un vídeo promocional para una startup o un anuncio para un nuevo producto.
ANIMACIONES:
Creo animaciones 2D y 3D que dan vida a ideas y conceptos, usando After Effects.
Por ejemplo: Animación de introducción para un canal de YouTube o para una escena de una serie de dibujos animados.
PROGRAMACIÓN:
En lo que a programación se refiere, soy capaz tanto de buscar soluciones ágiles a procesos costosos como de idear nuevos proyectos en diferentes lenguajes.
Por ejemplo: Un programa ágil en Phyton o el desarrollo de un nuevo videojuego.
OTROS INVENTOS:
Mis otros proyectos incluyen soluciones tecnológicas, prototipos, proyectos personales y experimentación con Arduino.
Por ejemplo: Un sistema de domótica para una vivienda.
